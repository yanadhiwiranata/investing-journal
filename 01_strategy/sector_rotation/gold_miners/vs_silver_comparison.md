## When to Enter Gold vs Silver

| Situation | Gold Miners ✅ | Silver Miners ✅ | Neither ❌ |
|-----------|---|---|---|
| **Inflation rising, yields falling, ISM >50** | STRONG BUY | STRONG BUY | — |
| **Inflation rising, yields stable, ISM 50–52** | BUY | PENDING ONLY | — |
| **Growth strong, ISM 52+, copper rallying** | ACCUMULATE | STRONG BUY | — |
| **Inflation sticky, yields rising, ISM declining** | AVOID | AVOID | ✓ Skip both |
| **Geopolitical spike, safe-haven mode** | ACCUMULATE | TAKE PROFITS (fade fast) | — |
| **Recession fears, ISM <48** | REDUCE POSITION | EXIT 100% | ✓ Skip both |

```

☐ Silver >$28?
☐ Auto/Solar demand stable?

```

**Key Difference:** Gold = macro play (5 min check). Silver = industrial demand play (15 min check).

## Position Sizing

| Metric | Gold | Silver |
|--------|------|--------|
| **Layer 1 Size** | 100 shares | 50–75 shares |
| **Layer 1 Discount** | Consensus –7% | Consensus –9% |
| **Position Weight Cap** | 35–40% | <25% |
| **Max Leverage to Commodity** | 1.5x–2.0x | 2.5x–3.0x |

**Why Different Sizes?**
- Gold miners: Steady, predictable leverage to gold
- Silver miners: 2.5–3.0x leverage to silver = much higher volatility
- Smaller position + smaller portfolio weight = risk management

## Decision Tree: Which One to Enter?

```
START: Is there a sector rotation opportunity?

├─ CHECK GOLD MACRO (Real yields, DXY, gold price)
│  ├─ Real yields falling + DXY weak → GOLD MINERS STRONG
│  ├─ Real yields rising + DXY strong → SKIP BOTH
│  └─ Real yields flat + DXY mixed → GOLD PENDING
│
├─ CHECK SILVER/INDUSTRIAL (ISM, Copper, Risk Sentiment)
│  ├─ ISM >51 + Copper up + Risk-on → SILVER MINERS STRONG
│  ├─ ISM <50 OR Copper down → SILVER MINERS SKIP
│  └─ ISM 50–51 + Mixed signals → SILVER PENDING
│
├─ WHICH ONE ENTERS?
│  ├─ BOTH bullish? → GOLD FIRST (macro-driven; more stable)
│  │                 Then add SILVER (if ISM confirms)
│  │
│  ├─ ONLY GOLD bullish? → Enter GOLD only
│  │
│  ├─ ONLY SILVER bullish? → Enter SILVER (but rare; requires industrial boom)
│  │
│  └─ NEITHER bullish? → WAIT (no forced entries)
│
└─ SIZING
   Gold entry: 100 shares Layer 1
   Silver entry: 75 shares Layer 1 (if also entering gold, may use 50)
   Never both maxed out; cap total miners weight <40% of portfolio
```

## Holding Periods Differ

| **CPI hotter than expected** | Likely +2–3% on real yields falling fear | Could spike initially but fade if recession fears |

## Memory Aid: The 3-Letter Rule

**GOLD:** GLD (ticker symbol)
- **G** = Growth concern? (If YES, avoid)
- **L** = Leverage to gold only? (If YES, pure play)
- **D** = DXY weakness? (If YES, bullish)

**SILVER:** SIL (silver miner ETF ticker)
- **S** = Supply shock? (If YES, bullish)
- **I** = Industrial demand? (If YES, strongest signal)
- **L** = Leverage 2.5–3.0x? (If YES, use 50–75 shares)
