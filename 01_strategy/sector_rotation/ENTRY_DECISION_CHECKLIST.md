# Gold vs Silver Miner Entry Decision Checklist

**Use this BEFORE placing any Layer 1 order. Takes 15–20 minutes.**

---

## STEP 1: MACRO REGIME CHECK (5 minutes)

### Real Yields Check (Gold Primary Driver)
```
[ ] Check FRED DFF10 for latest 10Y real yield
    Current: __________
    Trend: Rising / Stable / Falling
    
VERDICT:
□ Real yields <+0.5% AND stable/falling   → Continue to Step 2
□ Real yields +0.5 to +1.0% AND rising     → CAUTION (may skip gold)
□ Real yields >+1.0% AND rising sharply    → AVOID BOTH (wait for peak)
```

### DXY Check (Gold Secondary Driver)
```
[ ] Check TradingView DXY for latest level
    Current: __________
    24H change: + _______ or – _______
    
VERDICT:
□ DXY <103 AND falling                     → Bullish for gold
□ DXY 103–104 AND stable                   → Neutral
□ DXY >104 AND rising                      → Headwind for gold
```

### Gold Price Check
```
[ ] Check TradingView for spot gold
    Current: __________
    
VERDICT:
□ Gold >$2,300                             → Acceptable entry zone
□ Gold $2,250–$2,300                       → Entry only if other factors strong
□ Gold <$2,250                             → SKIP (support breaking)
```

**Gold Miners Entry Gate:**
- ✅ Real yields <+0.5%, DXY weak, gold >$2,300 → **PROCEED to GOLD ASSESSMENT**
- ⚠️ Real yields 0 to +0.5%, DXY mixed → **GOLD PENDING (wait for clarity)**
- ❌ Real yields rising, DXY strong, gold weak → **SKIP GOLD (wait for macro reset)**

---

## STEP 2: INDUSTRIAL DEMAND CHECK (10 minutes — For Silver Only)

### ISM Manufacturing Check (Silver Primary Driver)
```
[ ] Check FRED ISM PMI or Trading Economics for latest release
    Latest Release: __________ (date)
    ISM Level: __________
    Prior Month: __________
    Trend: Accelerating / Stable / Decelerating
    
VERDICT:
□ ISM >51 AND accelerating                 → STRONG (silver bullish)
□ ISM 50–51 AND stable                     → MARGINAL (caution)
□ ISM <50                                  → BROKEN (skip silver entirely)
□ ISM <48                                  → COLLAPSE (exit any positions)
```

### Copper Price Check (Silver Demand Canary)
```
[ ] Check TradingView copper (HG) for latest price
    Current: __________
    5-day change: + _______ or – _______
    
VERDICT:
□ Copper stable or +1%                     → Demand OK
□ Copper down 1–2%                         → Caution; monitor closely
□ Copper down >2%                          → Demand warning (cancel silver)
□ Copper down >3%                          → DEMAND BROKEN (exit silver)
```

### Auto/Solar Production Data Check
```
[ ] Check latest auto sales reports (US, China, EU)
    Status: Growing / Stable / Declining
    
[ ] Check solar module shipment trends
    Status: Growing / Stable / Declining
    
VERDICT:
□ Auto + Solar both stable/growing         → OK for silver
□ Either showing weakness                  → CAUTION; monitor
□ Guidance cuts announced                  → EXIT silver if in position
```

### Risk Sentiment Check
```
[ ] Check VIX level
    Current: __________
    
[ ] Check tech/crypto trends
    Status: Strong / Stable / Weak
    
VERDICT:
□ VIX <20 AND tech/crypto strong           → Risk-on (silver friendly)
□ VIX 20–25 AND mixed signals              → Neutral
□ VIX >25 AND tech weak                    → Risk-off (avoid silver)
```

**Silver Miners Entry Gate:**
- ✅ ISM >50, copper stable, auto/solar OK, risk-on → **PROCEED to SILVER ASSESSMENT**
- ⚠️ ISM 50–51, copper down 1%, mixed signals → **SILVER PENDING (wait for PMI/ISM confirmation)**
- ❌ ISM <50 OR copper down >2% OR risk-off → **SKIP SILVER (wait for demand reset)**

```
    Rating 5: __________ (date)
```

```
    Rating 1: __________ (date)
    Rating 2: __________ (date)
    Rating 3: __________ (date)
    Rating 4: __________ (date)
    Rating 5: __________ (date)
    
    Consensus: Buy / Hold / Sell
    Avg Price Target: __________
    Current Price: __________
    Upside/(Downside): __________ %
    
VERDICT:
□ Consensus Buy + Target >12% upside       → OK to enter (higher bar due to volatility)
□ Consensus Hold OR Target <8% upside      → SKIP
□ Consensus Sell                           → AVOID (downgrading)
```

---

## STEP 4: EARNINGS QUALITY CHECK (3 minutes)

```
[ ] Latest Quarterly Earnings Date: __________
    Days Since Release: __________ (if >21 days, get fresh analyst data)
    
[ ] Revenue vs. Guidance: On-track / Miss / Beat
[ ] AISC Costs: Stable / Improving / Deteriorating  
[ ] Production Guidance: On-track / Raised / Lowered
[ ] Dividend/Buyback: Maintained / Cut / Increased
[ ] Management Commentary: Confident / Cautious / Concerned

VERDICT:
□ All metrics on-track OR improving        → Quality OK (proceed)
□ Any metric deteriorating (AISC, production) → CAUTION (use larger discount)
□ Guidance cut or miss                     → SKIP (thesis broken)
```

---

## FINAL DECISION: WHICH TO ENTER?

```
BOTH macro gates passed (gold + silver)?
    YES → Enter GOLD first (100 shares Layer 1)
          Then evaluate SILVER separately (50–75 shares Layer 1 if demand strong)
    
    NO  → Only GOLD gate passed?
            YES → Enter GOLD only (100 shares Layer 1)
            
          Only SILVER gate passed?
            YES → Enter SILVER only (75 shares Layer 1, rare setup)
            
          NEITHER gate passed?
            NO → WAIT (no forced entries; patience wins)
```

## POST-ENTRY MONITORING

### Gold Miners (Weekly Check)
```
Every Friday EOD, check:
[ ] Real yields still <+0.5%? (YES = continue; NO = trim 25%)
[ ] Gold still >$2,300? (YES = OK; NO = raise stop)
[ ] Analyst consensus unchanged? (YES = OK; NO = reassess)
[ ] Position up +20%? (YES = take 50% profits)
```

### Silver Miners (More Frequent Check)
```
Every 2–3 days, check:
[ ] ISM trend intact? (YES = OK; NO = exit 50%)
[ ] Copper not down further? (YES = OK; NO = cancel Layer 2)
[ ] Auto/Solar data OK? (YES = OK; cuts announced = exit 50%)
[ ] Position up +15%? (YES = take 50% profits)
```
